# Brand Asset Naming Convention — v1

## Pattern
```
product_asset_variant_version.ext
```

## Fields

| Part | Values |
|------|--------|
| product | `vicelab`, `vibeguard`, `cookedpilot` |
| asset | `logo`, `icon`, `favicon`, `avatar`, `app-icon` |
| variant | `colour`, `black`, `white`, `dark`, `light` |
| version | `v1`, `v1.1`, `v2` |

## Examples
```
vicelab_logo_colour_v1.svg
vicelab_logo_black_v1.svg
vicelab_favicon_32x32_v1.png
vicelab_favicon_180x180_v1.png
vicelab_app-icon_512x512_v1.png
vicelab_avatar_colour_400x400_v1.png
cookedpilot_logo_colour_v1.svg
vibeguard_avatar_dark_400x400_v1.png
```
