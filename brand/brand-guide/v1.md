# ViceLab Brand System — v1
> Version: v1 | Status: LOCKED | Date: 2026-04-18

---

## Ecosystem Overview

ViceLab is a festival safety intelligence ecosystem with three products, each with a distinct identity, audience, and tone. All products share a cyberpunk-adjacent design language: dark backgrounds, gradient strokes, tight geometry.

---

## Brand Colors

|------|-----|---------|

**Gradient:** `linear-gradient(90deg, #0044CC 0%, #00AAFF 100%)`

## Naming Convention

Pattern: `product_asset_variant_version.ext`

Examples:
- `vicelab_logo_colour_v1.svg`
- `cookedpilot_favicon_32x32_v1.png`
- `vibeguard_avatar_dark_400x400_v1.png`

---

## Logo Usage Rules

| Context | Size |
|---------|------|
| Header / Nav | `height: 36px`, width auto |
| Product Hero | `height: 80–120px`, width auto |
| Favicon | 16×16, 32×32, 180×180 |
| App Icon | 512×512 |
| Social Avatar | 400×400 |

- **Never** stretch or distort
- **Never** mix product logos in one section
- **Always** `height` + `width: auto`
- **Always** root path `/logos/filename.svg`
